# Cyber Range Security Lab

## Overview
This repository documents hands-on work performed in a **Cyber Range environment** that simulates a real-world enterprise security infrastructure. The lab provides integrated security tooling and full administrative control over virtual machines, enabling practical experience in **security monitoring, threat detection, vulnerability management, and incident response**.

The goal of this project is to build job-ready skills aligned with **SOC Analyst** responsibilities using industry-standard platforms.

---

## Environment Architecture
The Cyber Range is a fully virtualized environment designed to replicate modern organizational security operations. It allows users to deploy, configure, and manage their own virtual machines while integrating enterprise security solutions for monitoring and detection.

Key capabilities include:
- Full administrator access to Windows and Linux virtual machines
- Security telemetry collection and log analysis
- Safe testing of detection, monitoring, and response workflows

---

## Security Platforms & Tools

### Microsoft Sentinel (SIEM)
- Cloud-native SIEM used for centralized log collection and analysis
- Configured to ingest endpoint and system telemetry
- Used to create analytics rules and investigate security incidents
- Experience aligns with enterprise SIEM platforms such as Splunk and QRadar

### Microsoft Defender for Endpoint (EDR)
- Endpoint Detection and Response platform
- Provides endpoint visibility, alerting, and threat investigation
- Integrated with Microsoft Sentinel for incident correlation

### Tenable (Vulnerability Management)
- Enterprise vulnerability scanning and assessment tool
- Used to identify, prioritize, and analyze system vulnerabilities
- Supports remediation planning and risk-based decision making
